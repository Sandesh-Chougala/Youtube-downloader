About Sandy's Downloader

Sandy's Downloader is a simple, clean, and fast tool that lets you download YouTube audio and video files with ease.
Built using Python (Flask) and powered by yt-dlp, it provides high-quality MP3 and MP4 downloads from YouTube links.

This is a personal project created by Sandesh for educational and learning purposes.
The tool is lightweight, privacy-friendly, and does not store any downloads.
